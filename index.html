Argument

Arguments are the values that are passed while calling a function

Parameter

Parameters are the variables declared during function definition.

Callback Function

Callback function is such a function which is passed as argument while invoking a function. Callback function is called inside the invoking function.

Synchronous

Synchronous request is such an request that cannot be carried out without the completion of previous event. Browser blocks other other request in this case. Single task is completed at once. Nested function is an example

Asynchronous

Asynchronous request is such a request that can be carried out along with other request pending in background. The browser is responsive in this case. Multitasking is possible. SetTimeOut() function is an example.

Anonymous Function

Anonymous function is a function with no name. It is mostly used in a scenario where function is called and defined at once. It can be used inside addEventListener method.

Hoisting

Hoisting is a JavaScript technique which moves variables and function declarations to the top of their scope before code execution begins
